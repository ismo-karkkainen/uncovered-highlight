# uncovered-highlight

A package and tool to display source code with uncovered parts highlighted.

## Note from a Human

I realized I wanted something like this tool that displays the parts of the source code uncovered by tests. Reports output to terminal show lines folowed by three dots when there are too many of them. I also did not care about the lines. If a condition is tested, but following block never covered, I want to know that.

I have no particular interest in making such a tool, and I may have missed one if there is something that does what I needed. Partially as a way to play around with generative AI code generation and agents, this package exists. As long as it works, I have no particular desire to touch the code, as long as I can help myself. I wanted the tool, I prompted, I got what you see.
